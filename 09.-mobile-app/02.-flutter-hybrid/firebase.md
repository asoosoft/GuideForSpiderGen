# (심화) Firebase 네이티브 푸시알림

> 네이티브 푸시를 통해 앱이 완전히 종료된 상태더라도 서버 신호를 통해 필요한 정보를 기기에게 전달할 수 있습니다.

### 1. flutter 플러그인 설치

플러터에서&#x20;

```powershell
cd 프로젝트이름
#프로젝트 디렉토리로 이동
flutter pub add firebase_core
#파이어베이스 sdk 기본 플러그인
flutter pub add firebase_messaging
#파이어베이스 메세지 기능을 위한 플러그인
```
