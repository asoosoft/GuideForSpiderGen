# 08. MessageBox

**AMessageBox**는 ADialog를 확장하여 메시지 박스를 띄우는 기능을 제공합니다.\
이 컴포넌트를 사용하면 사용자에게 메시지를 표시하고, 다양한 버튼 옵션을 통해 사용자의 입력을 받을 수 있습니다.
