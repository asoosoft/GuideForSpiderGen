# 4-1. View 로 보내기(TLAY)

### 1. Tlay 생성 <a href="#id-1.-component" id="id-1.-component"></a>

* Tlay 란?&#x20;
  * 스타일이 적용된 컴포넌트들을 그룹화하고, lay 에 배치하기 편리하게 관리하기 위한 layout.
* Spidergen 을 실행하고 기존 프로젝트를 열거나 새 프로젝트를 생성합니다.
* tlay 파일을 생성합니다.
  *

      <figure><img src="../../.gitbook/assets/image (294).png" alt=""><figcaption></figcaption></figure>
* 생성된 파일을 선택후 오른클릭후 편집모드로 열기를 실행합니다.  (Open As Layout)
  *

      <figure><img src="../../.gitbook/assets/image (295).png" alt=""><figcaption></figcaption></figure>
*





## 2. Instance 생성

* Figma 에서 생성한 Component 의 Instance 를 생성합니다
* [3.-css.md](3.-css.md "mention") 를 참조하여 컴포넌트를 생성합니다.
  *

      <figure><img src="../../.gitbook/assets/image (296).png" alt=""><figcaption></figcaption></figure>


* Instance 를 생성하기 위해 Assets 탭에서 Created In this file 을 선택합니다
  *

      <figure><img src="../../.gitbook/assets/image (297).png" alt=""><figcaption></figcaption></figure>


* 방금 프레임을 생성한 화면을 선택합니다. (여기서는 layout)
  *

      <figure><img src="../../.gitbook/assets/image (298).png" alt=""><figcaption></figcaption></figure>


* layout 화면의 컴포넌트들을 방금 생성한 Frame 쪽으로 드래그하여 Instance 를 생성합니다
  *

      <figure><img src="../../.gitbook/assets/image (299).png" alt=""><figcaption></figcaption></figure>


* 다시 File 탭으로 돌아와서 Frame 을 선택한후 플러그인을 실행합니다
  *

      <figure><img src="../../.gitbook/assets/image (301).png" alt=""><figcaption></figcaption></figure>


* View 로 보내기를 클릭하여 SpiderGen 에 컴포넌트 생성되었는지 확인.
  *

      <figure><img src="../../.gitbook/assets/image (302).png" alt=""><figcaption></figcaption></figure>


* Save All 로 생성된 화면 저장.
  *

      <figure><img src="../../.gitbook/assets/image (303).png" alt=""><figcaption></figcaption></figure>
